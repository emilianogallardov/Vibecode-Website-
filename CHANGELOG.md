# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-08-27

### Added
- Initial release of Vibecode Template
- React 18 with TypeScript and Vite
- Tailwind CSS v3 with dark mode support
- Edge API routes with Vercel deployment
- Production-ready security headers with CSP Report-Only
- Type-safe shared contracts with Zod
- Component structure with core/app separation
- ESLint v9 flat config with canonical TypeScript setup
- Vitest for testing with jsdom environment
- Path aliases (@/) for clean imports
- Vendor code splitting for optimized caching
